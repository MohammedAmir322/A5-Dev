Question-1: What are the different ways to select an element in the DOM?

getElementById: Select by ID. getElementsByClassName: Select elements by class (returns HTMLCollection). getElementsByTagName: Select by tag name (e.g., div). querySelector: Select the first match using CSS selectors. querySelectorAll: Select all matches (returns NodeList).

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Renders HTML content inside the element (includes tags). innerText: Shows visible text (ignores hidden elements, respects CSS). textContent: Shows all text (including hidden text, ignores HTML tags).

Question-3: What is event delegation in the DOM?

Attach an event listener to a parent element instead of individual child elements. Uses event bubbling to handle events from dynamic children (e.g., buttons added later).

Question-4: What is event bubbling in the DOM?

When an event on a child element "bubbles up" through its parent elements in the DOM hierarchy. Use event.stopPropagation() to stop it.

Question-5: How do you create, add, and remove elements using JavaScript?

Create: const el = document.createElement('div');
Add: parent.appendChild(el); or parent.insertBefore(el, referenceElement);
Remove: el.remove(); or parent.removeChild(el);
